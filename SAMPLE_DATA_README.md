## 🛠️ 프로토타입 구현 접근법

### 옵션 1: 규칙 기반 파싱 (빠른 검증용)
```python
# 정규표현식으로 "항목명 + 숫자 + 단위" 패턴 추출
pattern = r'([\w가-힣]+)(?:은|는|:)?\s*([0-9,]+(?:\.[0-9]+)?)\s*(억원|조원|%|원)'
```
- **장점**: 빠르고 예측 가능, 디버깅 쉬움
- **단점**: 복잡한 문장 구조 처리 어려움, 유지보수 필요

### 옵션 2: LLM 기반 파싱 (Claude API 활용) ⭐ 권장
